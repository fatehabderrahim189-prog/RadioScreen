# Dataset Strategy

## Current status

RadioScreen is an educational research prototype.

The current proof of concept uses only synthetic chest X-ray images generated
for software engineering and algorithm demonstration purposes.

No real patient data is included in this repository.

## Future work

Future research should use properly licensed public medical datasets, with
appropriate ethical approvals where required.

All experiments should be reproducible and clearly documented.

## Privacy

No identifiable patient information should ever be committed to this
repository.

Only legally obtained and appropriately licensed datasets may be used.
